# 💰 SpendWise – Smart Expense Tracker

SpendWise is a modern, cross-platform expense tracking application built with Flutter. It helps users manage daily expenses, track budgets, and visualize spending habits with clean analytics and a smooth UI experience.

---

## 📱 APK / Live App
👉 Installable APK: https://github.com/ankitky30-tech/spendwise/releases  
👉 Source Code: https://github.com/ankitky30-tech/spendwise

---

## 🚀 Features

- ➕ Add, edit, and delete expenses
- 🏷️ Category-based expense tracking
- 💰 Monthly budget management per category
- 📊 Interactive analytics (charts & breakdowns)
- 📉 Daily and monthly spending insights
- 📤 Export data to CSV and PDF reports
- 💾 Offline storage using Hive database
- 🎨 Clean, modern Material 3 UI design

---

## 🧠 Key Highlights

- Built with scalable Flutter architecture
- State management using Provider
- Optimized local storage with Hive
- Responsive UI for all screen sizes
- Real-world financial tracking logic implementation

## 🛠️ Tech Stack
